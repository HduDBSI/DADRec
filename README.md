# DADRec Beauty Test

This package keeps only the PreOPQ/SoftQuant DADRec Beauty test artifacts.

- Model: `DADRec`
- Checkpoint: `ckpt/beauty.pth`
- Cache: `cache/AmazonReviews2014/Beauty`
- Test script: `test_beauty.sh`

Run:

```bash
bash test_beauty.sh
```

Set `PYTHON_BIN=/path/to/python` if the default Python does not have the required packages.
